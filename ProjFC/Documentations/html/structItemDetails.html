Detailed Description

Stores the name and description for some item type that can be carried by player characters as part of their inventory.

The 'name' and 'desc' fields must contain a valid, NUL-terminated string of length at most DEFAULT_BUFFER_SIZE-1, and should otherwise comply with the requirements laid out in the project specification for valid ItemDetails structs.
